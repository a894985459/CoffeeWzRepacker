## Features / 功能總覽

<details>
<summary>Click to expand / 點擊展開</summary>

### File & Data / 檔案支援
- Support WZ / IMG / MS / PKG2 parsing  
  支援 WZ / IMG / MS / PKG2 解析  
- Load entire folders  
  支援整個資料夾載入  
- Version auto-matching  
  自動匹配檔案版本  

---

### Editor / 編輯器
- Dark / Light themes  
  深色 / 淺色模式  
- Custom hotkeys  
  自訂快捷鍵  
- Anchor visualization (lt / rb)  
  錨點顯示  

---

### Animation / Spine
- Spine 2.1.27 / 4.1.24 support  
  支援 Spine 動畫  
- Export PNG / GIF  
  匯出 PNG / GIF  
- Frame extraction  
  單幀提取  

---

### Tools / 工具
- Recursive search  
  遞迴搜尋  
- Multi-node editing  
  多節點編輯  
- Batch image import  
  批次匯入圖片  

</details>

## Detailed Features / 詳細功能

<details>
<summary>Click to expand / 點擊展開</summary>

01. Automatic file version matching for IV alignment analysis  
    自動匹配檔案版本  

02. Supports analysis up to KMST 1.2.1199  
    支援解析至 KMST 1.2.1199  

03. Supports .ms and PKG2 files  
    支援 .ms 與 PKG2  

04. Customisable hotkeys and UI colors  
    自訂快捷鍵與UI顏色  

05. Recursive search across files  
    遞迴式搜尋  

06. Tab management improvements  
    分頁功能優化  

07. Side-by-side file comparison  
    支援雙視窗對照  

08. Image zoom (20% ~ 1600%)  
    圖片縮放  

09. Video parsing (PNG / GIF export)  
    影片解析  

10. Spine animation parsing & export  
    Spine 動畫支援  

11. Auto image linking for Canvas nodes  
    Canvas 自動補圖  

12. Support for anchors (lt / rb)  
    錨點支援  

13. PNG format support (DXT / BC7 etc.)  
    支援多種壓縮格式  

14. Folder loading support  
    支援資料夾載入  

15. Node expansion & collapse  
    節點展開收合  

16. Multi-canvas animation playback  
    多選動畫播放  

17. Quick repeat node (F4)  
    快捷重複操作  

18. World map editing  
    世界地圖編輯  

19. Skill editing tools  
    技能編輯  

20. Batch image import to nodes  
    批次圖片匯入  

</details>
